# Turborepo Monorepo

This is a basic Turborepo monorepo setup with `apps` and `packages` folders.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run a turbo command (e.g., build):
   ```sh
   npx turbo run build
   ```

## Structure
- `apps/` – Application projects
- `packages/` – Shared packages

See [https://turbo.build/repo/docs](https://turbo.build/repo/docs) for more info.
